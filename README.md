# PostApo RP | 1.20.1 Minecraft Forge - White Eagle PMC
Oficjalny Mod pack serwera Post Apocaliptic RP



od teraz nie będziecie mogli grać na serwerze bez zainstalowanej lokalnie paczki modów
modding tutorial dla osób które jeszcze tego nie robiły albo zapomniały:
1. pobieramy sobie [`Forge 1.20.1 - 47.4.10 `](https://adfoc.us/serve/sitelinks/?id=271228&url=https://maven.minecraftforge.net/net/minecraftforge/forge/1.20.1-47.4.10/forge-1.20.1-47.4.10-installer.jar) jako `installer`
2. instalujemy Forge jako `client`
3. pobieramy cały folder z GitHub -a
4. w wyszukiwarce windows wpisujemy `%appdata%` aby przejść do lokalnego folderu, w którym zainstalowany jest Minecraft
5. wchodzimy w `/.minecraft/mods`
6. tu wypakowujemy zawartość pobranej wcześniej paczki
Po zakończonym modowaniu ścieżka do pliku dowolnego moda powinna wyglądać następująco:
```C:/Users/[user]/AppData/Roaming/.minecraft/mods/[mod]```
